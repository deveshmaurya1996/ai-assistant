# Specialized Agents

Multi-agent packages for the cognitive runtime supervisor.

| Agent | Role |
|-------|------|
| [planner](planner/README.md) | Task decomposition, capability selection |
| [memory](memory/README.md) | Retrieval and memory writes |
| [browser](browser/README.md) | Playwright automation |
| [communication](communication/README.md) | Email, WhatsApp, messaging |
| [safety](safety/README.md) | Risk and permission checks |
| [reflection](reflection/README.md) | Self-correction and verification |
| [voice](voice/README.md) | Realtime and classic voice |

Implementation status: stubs. Production logic currently lives in `services/cognitive-runtime` (formerly `ai-orchestrator`).
